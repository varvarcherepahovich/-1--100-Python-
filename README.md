# 🎲 Guess the Number (Desktop Edition)

Простая и увлекательная игра "Угадай число", написанная на **Python** с графическим интерфейсом. Проект скомпилирован в `.exe`, что позволяет запускать его на Windows без установки интерпретатора Python.

## 🚀 Особенности
* **Диапазон:** Игра загадывает случайное число от **1 до 100**.
* **GUI:** Удобный графический интерфейс (никакой консоли).
* **Подсказки:** Программа сообщает, больше или меньше ваше число относительно загаданного.
* **Portable:** Весь проект упакован в один исполняемый файл `.exe`.

## 🛠 Технологии
* **Language:** Python 3.x
* **Library:** Tkinter / CustomTkinter (для интерфейса)
* **Build Tool:** PyInstaller (для конвертации в .exe)

## 📦 Как запустить
1. Перейдите в раздел [Releases](https://github.com/varvarcherepahovich/-1--100-Python-/releases/tag/python).
2. Скачайте архив с файлом `GuessTheNumber.exe`.
3. Запустите файл и попробуйте угадать число за минимальное количество попыток!

> **Математический факт:** Используя стратегию бинарного поиска, любое число от 1 до 100 можно угадать максимум за **7 попыток**. Проверь себя!
